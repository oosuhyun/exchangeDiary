# Introduce
여행마다 친구들과 교환일기를 만들고 거기서 일기와 댓글를 적어나가며 기억을 추억하는 기능과 6개월 뒤에 편지를 전달하는 느린 편지 기능을 갖춘 어플리케이션을 기획하였습니다.
저는 JPA, 팀원는 Mybatis를 사용하여 각자 백엔드를 개발해보는 것으로 목표 삼았으며, 이를 완수하였습니다.
